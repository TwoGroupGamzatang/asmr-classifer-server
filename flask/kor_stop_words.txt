2. user_preference 몇개 선택 -> 개인별 classifeir 생성 -> 아티클 url 입력 -> 개인별 classifier로 분류 -> 분류 결과 화면에 표시됨 

3. 사전에 가상의 유저 생성 : 이 유저는 기존에 피드백을 몇번 주어서 어느정도 개인화가 된 상태 -> 아티클 url 입력 -> 분류 결과 화면에 표시됨 -> 차이 비교
