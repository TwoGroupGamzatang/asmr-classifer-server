# ASMR Test Server
Code for testing ASMR service by using Rest API

### 로컬 환경에서 실행 시 
- python3 설치 
- flask/app.py의 필요한 패키지 설치

### 가상환경에서 실행 시
```bash
$ pip install pipenv
```
flask 폴더에서 실행
```bash
$ pipenv shell
```
flask 폴더 내에 Pipfile 보고 필요한 패키지 자동 설치
```bash
$ pipenv install
```

### 실행 환경
- vscode의 live server extension
